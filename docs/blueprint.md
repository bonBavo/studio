# **App Name**: Engineer's Workshop Portfolio

## Core Features:

- Interactive Garage Interface: Presents the portfolio as an interactive garage interface, with projects displayed as cars.
- Command Palette: Implements a command palette for easy navigation.
- AI-Powered Career Path Tool: Offers personalized recommendations on which languages and tools a developer should learn next, given their career goals and interests. The AI tool can assess a developer's skill set and future preferences to suggest tailored educational resources.
- Garage Theme Customization: Allows users to switch between different 'garage' themes, like a race garage, a vintage garage, or a futuristic garage, complete with different tool and car styles.
- Automated Job Application Insights: Using the submitted resume data and accessed job posting data, this generative AI tool identifies key criteria and insights for relevance ranking against roles, which informs improvements and highlights transferable skills for use in the portfolio to impress recruiters.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A), evoking the serious yet intriguing atmosphere of a garage or workshop setting.
- Background color: Very dark grey (#212529), providing a strong contrast against the primary and accent colors, akin to the dim lighting of a garage.
- Accent color: Bright Orange (#EA580C), inspired by the colors often used for tools and details in garages.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and shorter content to match the tech theme; body font: 'Inter' (sans-serif) for comfortable readability in longer texts.
- Use tool-themed icons to represent skills and technologies.
- Garage or workshop-inspired layout, organizing elements in the style of tools on a wall, or cars in the garage.
- Subtle animations of tools and cars as users scroll through projects.
